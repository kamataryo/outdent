# outdent

[![Build Status](https://travis-ci.org/kamataryo/outdent.svg?branch=master)](https://travis-ci.org/kamataryo/outdent)

Outdent is a simple here document generator with pretty indents.

## install

```shell
$ yarn add outdent
# or
$ npm insall outdent
```
